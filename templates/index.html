Game-Onz Recommendation Engine

Welcome to Game-Onz Recommendation Engine. A website for gamers to recommend the games that will fit the user's interests, and then content-based filtering is applied to find similarities between them.

This website are created by using Content-Based Filtering approach and Cosine Similarity algorithm.

The Mobile Game and Computer Game in this website is limited since it is just a free dataset from kaggle.
